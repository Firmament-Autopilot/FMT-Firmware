name: cyapp_prov_oem
description: The application injects OEM assets and advances the device LCS to NORMAL_NO_SECURE or SECURE LCS. The OEM assets are SMIF configuration, OEM public key hashes, access restrictions, SWD listen window and NV counter. The valid device LCS to launch this application is NORMAL or NORMAL_NO_SECURE.
type: Firmware application
version: 1.3.1.4803
